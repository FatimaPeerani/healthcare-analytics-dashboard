# 🏥 Healthcare Analytics Dashboard

An interactive **Power BI dashboard** built to analyze and visualize healthcare data, featuring a professional dark theme and multi-dimensional insights.

![Dashboard Preview](Screenshot%202026-06-11%20033514.png)

---

## 📊 Dashboard Overview

This dashboard provides a comprehensive view of healthcare data across **55,500+ patient records**, covering medical conditions, admissions, billing, and more.

### KPI Cards
| Metric | Value |
|---|---|
| 👥 Total Patients | 55.5K |
| 🎂 Average Age | 51.54 |
| 💰 Average Billing Amount | $25.54K |

---

## 📈 Visualizations

- **Medical Condition Distribution** — Donut chart showing breakdown of Arthritis, Diabetes, Hypertension, Obesity, Cancer, and Asthma
- **Gender Distribution** — Pie chart showing near-equal Male/Female split (50/50)
- **Admission Type Breakdown** — Bar chart comparing Elective, Urgent, and Emergency admissions
- **Test Results Analysis** — Column chart showing Abnormal, Normal, and Inconclusive results
- **Billing Trend by Year** — Line chart showing billing amount trends from 2019–2024
- **Insurance Provider Analysis** — Treemap showing billing by Cigna, Blue Cross, UnitedHealth, Medicare, and Aetna
